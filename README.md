# google-home-pokemon-dictionary
Pokemon dictionaryn Google Home

## how to run
install requires

```sh
# apt-get install libnss-mdns libavahi-compat-libdnssd-dev
```

install node modules

```sh
$ yarn install
$ yarn sed
```

run

```sh
$ yarn start
```
